# PSL Finance Hub SaaS App

A real React SaaS app for PSL Finance Hub.

## Features
- Landing page
- Login/signup screen
- Investor dashboard
- Lead search and filtering
- CSV export
- Admin panel
- Subscription upgrade link
- Supabase-ready setup

## Local setup
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Upload this folder to GitHub.
2. Import the repository into Vercel.
3. Add environment variables from `.env.example`.
4. Deploy.
